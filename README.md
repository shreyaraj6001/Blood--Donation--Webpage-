# 🩸 Blood Donation Website

## 📌 Project Description
The Blood Donation Website is a simple and user-friendly platform designed to help people find blood donors quickly and easily. This website allows users to register as donors and helps patients or their families search for available donors when needed.

This project aims to support emergency situations and promote voluntary blood donation.

---

## 🚀 Features
- Donor Registration Form  
- Search Blood Donors  
- User-friendly Interface  
- Fast and Simple Navigation  
- Responsive Design  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## 🎯 Purpose of Project
This project is created to help people in emergency situations find blood donors quickly and efficiently. It also encourages more people to participate in blood donation.
